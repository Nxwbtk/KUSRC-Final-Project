# KUSRC-Final-Project

## Setup Instructions

1. Initialize the virtual environment:

   ```sh
   python -m venv venv
   ```

2. Activate the virtual environment:

   ```sh
   .\venv\Scripts\activate
   ```

3. Run the server:

   ```sh
   python socket-api.py
   ```

4. Run the camera:
   ```sh
   python cam.py
   ```
